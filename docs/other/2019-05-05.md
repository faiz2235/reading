# 我的生活不可能那么坏
这是一本漫画集，由日本的一位插画师河尻圭吾所著。

漫画中将一些日常生活的小细节放大后，冷笑话混合着黑色幽默，仿佛是生活开的玩笑，不管你愿不愿意，都要陪着它一起苦笑。有意思。
